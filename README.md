# Max Techera

Building autonomous agent infrastructure. Montevideo, Uruguay.

I build open-source tools that make AI agents reliable — dispatch them, verify their output, and let them get smarter over time.

## What I'm Working On

### Agent Infrastructure

| Project | What It Does | Install |
|---------|-------------|---------|
| [**orchestrator**](https://github.com/maxtechera/orchestrator) | Dispatch agents, verify deliverables, improve with every cycle. Skill-first framework — your ticket board becomes an autonomous workforce. | `clawhub install orchestrator` |

## The Stack

**Agent platforms**: Claude Code, OpenClaw (Railway), Gemini CLI, Codex CLI

**How it fits together**: The orchestrator reads your ticket board and dispatches domain-skilled agents. Memory ensures nothing is forgotten between sessions. Skills are the domain expertise — install what you need. Everything is a text file agents read and follow.
