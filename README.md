## How It Fits Together

The **orchestrator** reads your ticket board and dispatches domain-skilled agents. **Memory** ensures nothing is forgotten between sessions. **Ship** validates every credential before deploy. **Skills** add domain expertise on demand.

Everything is a text file agents read and follow. No DSLs. No wrappers. Just prompts and Python.

**Platforms**: Claude Code · OpenClaw · Gemini CLI · Codex CLI · ClawHub
